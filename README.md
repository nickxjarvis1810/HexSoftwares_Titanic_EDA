# HexSoftwares_Titanic_EDA
EDA on Titanic Dataset using Python

Titanic EDA Project

Project Description

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify patterns affecting passenger survival.

🛠️ Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

Key Steps

* Data Cleaning (handling missing values)
* Data Analysis
* Data Visualization

Insights

* Female passengers had higher survival rates
* 1st class passengers had better survival chances
* Age had a moderate impact on survival

Dataset

Titanic dataset containing passenger information such as age, class, fare, and survival status.
